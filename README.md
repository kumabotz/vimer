## TIPS
- `f` find in line
- `F` find backward in line
- `vt<CHAR>` select until character
- `vi<CHAR>` select inside character
- `va<CHAR>` select around character (character is included)
- `gc` comment line
- `gcgc` uncomment line
- `m<CHAR>` mark current line
- `<backtick><CHAR>` go to mark
